# Bulk RNA-seq Analysis for "Interaction between TREM2-Macrophages and Cutibacterium acnes Drives Altered Lipid Metabolism in Chronic Apical Periodontitis"

This repository contains the custom R scripts and processed count data used for the bulk RNA-sequencing analysis in the manuscript: 
**"Interaction between TREM2-Macrophages and Cutibacterium acnes Drives Altered Lipid Metabolism in Chronic Apical Periodontitis"** by Dawool Han, Nam Hee Kim, et al.

## Repository Structure

To ensure reproducibility, this project uses the `here` package. Please maintain the following directory structure when running the scripts:

```text
├── README.md
├── sessionInfo.txt
├── scripts/
│   ├── 01_RNAseq_Fig3_FigS5b.R
│   └── 02_RNAseq_FigS5a.R
├── input/
│   ├── gene_count fig3 figS5b.xls
│   └── gene_count figS5a.xls
├── plot/          # Generated figures (.tiff) will be saved here
└── output/        # Generated supplementary tables (.xlsx) will be saved here
```

## File Descriptions
R Scripts
01_RNAseq_Fig3_FigS5b.R: Performs differential gene expression (DGE) analysis between C. acnes-infected and control macrophages using edgeR. Generates the heatmap (Row Z-score of log2(TPM+1)), volcano plot, and over-representation analysis (ORA) dot plots for KEGG and GO (BP, CC, MF) pathways.

02_RNAseq_FigS5a.R: Performs DGE analysis comparing CA_G2 and Control samples. Generates the corresponding heatmap, volcano plot, pairwise scatter plots, and sample correlation matrix based on Pearson correlation coefficients.

## Input Data
gene_count fig3 figS5b.xls: Raw gene count matrix generated by featureCounts for the Figure 3 and Figure S5b analysis.

gene_count figS5a.xls: Raw gene count matrix generated by featureCounts for the Figure S5a analysis.

## Dependencies & Environment
All statistical analyses were performed in the R statistical environment (v4.4.2). Key packages required to run the scripts include:

edgeR (v4.4.2)

clusterProfiler (v4.14.6)

org.Hs.eg.db (v3.20.0)

enrichplot (v1.26.6)

tidyverse (v2.0.0)

pheatmap (v1.0.12)

gplots (v3.3.0)

For a complete and exact list of attached packages and versions, please refer to the sessionInfo.txt file included in this repository.
